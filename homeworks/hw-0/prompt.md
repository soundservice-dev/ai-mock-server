## Goal
Создать Rails-приложение (mock-сервер) и реализовать первую фичу: создание простого мока вручную через веб-интерфейс.

## Context
- См. PROJECT.md и CLAUDE.md
- Это новый проект, Rails-приложения ещё нет

## Requirements
1. Сделать базовую аутентификацию
2. Создать CRUD для управления моками. QA при создании может указать:
  - url запроса
  - http-метод запроса
  - заголовки запроса
  - тело запроса
  - возвращаемый статус-код
  - заголовки ответа
  - тело ответа
3. Клиент мок-сервера (QA или внешняя система) может отправить запрос и получить ожидаемый ответ

## Acceptance
- [ ] rails new выполнен, БД настроена
- [ ] Можно создать мок через браузер (форма)
- [ ] Можно отправить запрос на замоканный URL и получить настроенный ответ
- [ ] Тесты проходят
